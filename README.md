# Space Invaders

### __Game Description__

Enjoy a rebuild of the classic game, Space Invaders, in all its glory! As the main player, shoot as many of the aliens approaching you as you can and complete the game.

Use the left and right arrow to move in those directions in the game window, and press Space to shoot lasers towards the aliens. Delete each of the aliens to win the game!

Have fun playing this desktop version of the game!

### __Technologies Used__
##### _HTML_
  The HTML code is used to house the: 
  * Game title
  * Game header
  * Game board linking

###### _CSS_ 
  The CSS code is used to house the styling of the:
  * Title font
  * Game board layout
  * Game background

###### _JavaScript_ 
  The JaveScript code is used to house the functions & logic of the:
  * Player movement, shooting and actions
  * Enemy aliens movement, shooting and actions
  * Game rendering
  * Game over scenario
  * Game speed

###### _GitHub_ 
  The GitHub repository is used to remotely contain:
  * HTML code and tagging
  * CSS styling code
  * JavaScript functionality for players
  * Player and enemy images  functionality
  * Player and enemy sounds for shooting and deletion
